# v0.7.0 - Pro-Router Enhancements

This major update transforms `nix-router-optimized` into an enterprise-grade routing stack, focusing on High Availability, Performance, and "OpenBSD-tier" Security.

## Highlights

### 🚀 High Availability (HA)
- **VRRP (Keepalived)**: Implemented `services.router-ha` for Virtual IP (VIP) failover between master and backup nodes.
- **Kea DHCP HA**: Extended Kea module with native load-balancing and lease synchronization support.
- **Smart WAN Failover**: Integrated "Golden MAC" cloning and interface management for seamless ISP handover when using an unmanaged switch topology.

### 🌐 Advanced Networking
- **Multi-WAN**: Refactored `router-networking` to support multiple ISP uplinks simultaneously.
- **Multi-WAN Monitor**: Added `services.router-mwan` for automatic health-checking and priority-based metric switching.
- **NAT64 / DNS64**: Added stateless NAT64 (Tayga) and synthesized DNS64 (Unbound) for IPv6-only internal transition.
- **SQM (Cake)**: Added Smart Queue Management using the CAKE queuing discipline to eliminate bufferbloat.

### 🛡️ Security & Hardening
- **Kernel Tuning**: Applied strict kernel parameter hardening (ASLR, dmesg restrictions, TCP stack hardening).
- **Geo-IP Blocking**: Added declarative, automated blocking of inbound traffic from specific countries via nftables.
- **MAC Security**: Implemented interface-specific MAC address whitelisting with enforcement and alert policies.
- **mDNS Reflector**: Safely repeat mDNS discovery across isolated VLANs (LAN, IoT, etc.).
- **UPnP/NAT-PMP**: Secure-by-default MiniUPnPd integration with nftables.

### 📊 Dashboard & Diagnostics
- **NAT64 Widget**: Real-time visualization of NAT64 prefix, pool, and active translations.
- **Service Monitoring**: All new "Pro" services are automatically detected and monitored by the dashboard.
- **API Enhancements**: New endpoints for real-time connection tracking and translation status.

## New Modules
- `router-ha`: VRRP and HA orchestration.
- `router-mwan`: Multi-WAN health monitoring.
- `router-sqm`: CAKE-based traffic shaping.
- `router-mdns`: Avahi-based mDNS reflection.
- `router-upnp`: MiniUPnPd integration.
- `router-nat64` & `router-dns64`: IPv6 transition tools.
- `router-security-hardened`: System-wide security profiles.

## Internal Changes
- Updated `router-networking` to handle dynamic MAC cloning and multiple WAN interfaces.
- Improved `router-kea` with high-availability hook libraries.
- Added comprehensive evaluation smoke tests in `tests/pro-features-smoke.nix`.
