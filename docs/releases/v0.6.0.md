# v0.6.0

## Added

- Declarative Technitium blocklist management via the new `dns-blocklists` module.
- Curated preset support for:
  - `hagezi-light`
  - `hagezi-normal`
  - `hagezi-pro`
  - `hagezi-pro-plus`
  - `hagezi-ultimate`
  - `hagezi-nrd-14d`
  - `oisd-big`
  - `stevenblack`
- Additive `extraUrls` support so hosts can append custom feeds without replacing the defaults.
- A generated `/etc/technitium/blocklists.json` artifact to make the active declarative selection visible on the host.

## Changed

- Gateway configuration now enables declarative Technitium blocklists with a conservative default set:
  - `hagezi-normal`
  - `hagezi-nrd-14d`

## Notes

- Technitium already ships a builtin quick-pick catalog in its web UI. This release does not replace that catalog; it makes the active blocklist selection reproducible in Nix.
- The sync service preserves the current Technitium blocking mode and related blocking settings, and only manages the configured blocklist URLs plus refresh interval.
